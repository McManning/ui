# Buckeye UX Components

This section contains components implemented to meet feature parity with the BUX design system for Drupal and Twig.

These are not typically used in Office of Research applications and are typically aimed toward marketing websites.

For more information on Buckeye UX, visit [bux.osu.edu](https://bux.osu.edu/).
